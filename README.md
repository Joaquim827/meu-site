# Site Curriculo

Portfolio pessoal de Joaquim, desenvolvido para apresentar perfil profissional, habilidades, projetos e formas de contato em uma pagina responsiva.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Layout responsivo para desktop e mobile
- Menu de navegacao adaptavel
- Animacoes suaves ao rolar a pagina
- Secao de projetos com previa do projeto publicado
- Links de contato e chamada para visualizar projetos

## Estrutura

```text
.
├── index.html
├── index.js
├── styles/
│   └── style.css
└── images/
```

## Como visualizar

Abra o arquivo `index.html` no navegador.

Tambem e possivel acessar a versao publicada pelo GitHub Pages, caso esteja habilitada no repositorio.
